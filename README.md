# Habit Tracker

A minimal habit tracking app built with React, Vite, and Tailwind CSS.

## Features

- Add and delete habits
- Mark habits as complete each day
- Track streaks automatically
- Dashboard showing total habits, daily progress, and top streak
- Data persisted in localStorage — no backend needed

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
